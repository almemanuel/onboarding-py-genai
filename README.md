# onboarding-py-genai

This repository is my preparatory work for new porjects related to GenAI. It includes code snippets, examples, and resources that I find useful for understanding and working with Generative AI technologies. The goal is to have a centralized place where I can quickly access information and tools needed for my GenAI projects.

## Dependency
- Python 3.8 or higher

## Install
1. clone this repository: `git clone https://github.com/almemanuel/onboarding-py-genai.git`
2. start the virtual environment: `python -m venv venv`
3. activate the virtual environment: `source venv/bin/activate` (Linux/Mac) or `venv\Scripts\activate` (Windows)
4. install the required packages: `pip install -r requirements.txt`
